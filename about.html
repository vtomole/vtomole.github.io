It is supposed to be the antithesis of functional programming languages which use the axioms of Lambda Calculus to do its' computations.

I am currently working on the project by myself. If more people are interested in helping me build the language, they are more than wecome to do so. This project has helped me learn a lot about how programming languages, interpreters, and compilers work.
