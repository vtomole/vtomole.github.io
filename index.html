Interplanetary File System in Common Lisp:

CL-IPFS is the IPFS protocol written in Common Lisp written for the Mezzano Operating System. The goal is to make the Operating System capable of having peer-to-peer file system that will be more resistant to cyber attacks and censorship.

This project has taught me a lot time management. Since it is very hard to balance my coursework and extra-curricular projects, I have learned that I can only focus on two projects at most if I ever want to get any studying done. This project has also taught me about networking and the infrastructure of the internet.

This project is different from how "regular" Operating Systems works. The way the internet works now is that requests are sent to and from servers. IPFS changes that by putting the internet in a peer-to-peer file system. When this protocol is combined with an Operating System, Applications load quicker and they work when the user is offline.

This project will be of interest employers, because it will show that I'm capable of building a network stack from the ground up.
